A technical sound design project exploring adaptive music, interactive ambience and parameter-driven audio implementation using FMOD Studio.

Project Demonstration

This demonstration showcases the implemented adaptive music system, dynamic ambience, environmental transitions and parameter-driven audio behaviours. The project received an overall mark of 83%.

About the Project

This project was created as part of my BSc Music Technology degree and focused on designing an interactive audio system for a conceptual top-down action game. Every sound effect, ambience layer and musical element was created before being implemented in FMOD Studio using non-linear playback and gameplay-driven parameters.

Development Process

Planning

Identified every required gameplay, UI, ambience and music asset before production.

Sound Design

Produced original sound effects and adaptive musical material using Ableton Live.

FMOD Implementation

Implemented events, parameters and non-linear playback systems within FMOD Studio.

Testing & Refinement

Balanced transitions, refined parameter behaviour and tested playback consistency.
