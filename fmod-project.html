Interactive Game Audio Implementation

Designed and implemented a complete interactive audio system for a conceptual top-down action game using FMOD Studio. The project focused on creating responsive music systems and immersive environmental audio through parameter-driven implementation, achieving an overall mark of 83% with strong academic feedback.

Overview

The objective of this project was to demonstrate how interactive audio can enhance player immersion through adaptive music and dynamic environmental sound. Every sound effect, ambience layer and music system was designed specifically for the project before being implemented within FMOD Studio.

Technical Features

  • Designed a complete audio asset list covering gameplay, UI, ambience and music.
  • Created adaptive battle music that increases in intensity according to player health, enemy count and game progression.
  • Implemented dynamic environmental ambience featuring randomised traffic, emergency vehicles and evolving weather conditions.
  • Designed location-based transitions including real-time filtering effects to simulate movement between environments.
  • Structured the project using parameter-driven events and non-linear playback techniques within FMOD Studio.

Software Used

  • FMOD Studio
  • Ableton Live
