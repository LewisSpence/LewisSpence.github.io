Audio Plugin Development • Digital Signal Processing • Stereo Imaging

A custom stereo widening audio plugin built to create immersive stereo fields through Haas-effect processing, mid/side control and spatial EQ.

Expansive Width is a DSP plugin developed to provide intuitive control over stereo image manipulation while preserving clarity and mono compatibility. Designed as both a creative sound-design tool and practical mixing processor, it combines independent Haas-effect delays, frequency shifting, mid/side processing and spatial filtering into a single workflow for widening instruments, vocals and ambience.

About the Project

Expansive Width was developed to explore practical stereo image enhancement through a clean, user-focused interface. Rather than relying on a single widening algorithm, the plugin combines multiple complementary spatial processing techniques into a modular signal chain, allowing width, depth and tonal balance to be adjusted independently.

The core widening engine is based on Haas-effect principles, using independently adjustable left and right delay times alongside subtle frequency shifting to increase perceived stereo width while maintaining a natural sound. A dedicated mid/side processing section allows the centre image to be reduced or emphasised independently of the side information, making it possible to create anything from subtle stereo enhancement to expansive, atmospheric soundscapes.

To complement the spatial processing, the plugin also incorporates an integrated filter section featuring multiple filter types, adjustable resonance and graphical EQ control, enabling tonal shaping without requiring external plugins. A final width stage provides additional stereo positioning and output control, making Expansive Width suitable for both creative production and mix refinement.

Throughout development, particular attention was given to workflow and usability. Parameters were grouped into logical processing stages, allowing users to understand the signal path quickly while encouraging experimentation through immediate visual feedback.
